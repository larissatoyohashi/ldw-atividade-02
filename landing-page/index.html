Sistema de Gestão de Residências

Aplicação full-stack para cadastro e listagem de propriedades. Arquitetura dividida entre backend (API e banco de dados) e frontend (Interface de usuário).

1. Execução do Backend

O backend deve ser inicializado antes do frontend para garantir o fornecimento correto dos dados.

  1. Abra o terminal e acesse o diretório do backend: 
    cd backend
  2. Instale as dependências diretamente via pip: 
    pip install fastapi uvicorn
  3. Inicie o servidor local: 
    uvicorn main:app --reload

O servidor estará rodando em http://127.0.0.1:5000

2. Execução do Frontend

Certifique-se de que a API (backend) esteja operando antes de iniciar a interface.

  1. Abra um novo terminal e acesse o diretório do frontend: 
    cd frontend
  2. Instale as dependências da interface e conexão: 
    pip install flet requests
  3. Execute a aplicação: 
    python main.py

A interface será aberta automaticamente no seu navegador padrão.
