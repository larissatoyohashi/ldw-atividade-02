# ldw-atividade-02
# Sistema de Gestão de Residências

Este projeto consiste em uma aplicação full-stack para o cadastro e a listagem de propriedades residenciais. A arquitetura é dividida em uma API (Backend) e uma interface de usuário (Frontend), ambas desenvolvidas em Python.

---

## Tecnologias Utilizadas

* **Backend:** Flask, Flask-CORS, Flasgger (documentação Swagger) e Pydantic (validação de dados).
* **Frontend:** Flet (interface gráfica) e Requests (comunicação HTTP).
